Invertebrates

Quantifying mobile invertebrate communities

Introduction

Mobile invertebrates are key components of marine ecosystems, performing vital ecological functions and serving as indicators of ecosystem health. The Pristine Seas invertebrate survey methodology provides standardized protocols for quantifying the abundance, size distribution, and diversity of conspicuous mobile invertebrates across different habitats and depths. These surveys complement our benthic composition data (LPI) by documenting the mobile fauna that contribute to reef processes such as herbivory, predation, and bioerosion.

Overview

The invertebrate survey method involves systematically searching a defined belt transect area to identify, count, and measure key mobile invertebrate species. This approach provides quantitative data on invertebrate density, biomass, and community composition, enabling rigorous spatial and temporal comparisons.

Key Features

  • Defined search area: 50 m × 1 m belt transect (50 m²)
  • Taxonomic focus: Conspicuous echinoderms, mollusks, crustaceans, and other mobile invertebrates
  • Size estimation: Measurements of key species (e.g., giant clams, trochus snails)
  • Standardized effort: Consistent search time and methods across stations
  • Integration: Same transect line as LPI and recruitment surveys

Field Implementation

The invertebrate survey is typically conducted by the recruit/invert diver as part of the three-person benthic survey team. This specialized diver completes both the invertebrate and coral recruit surveys during a single dive, performing these tasks in sequence.

Invertebrate Survey Specifications
  • Transect dimensions: 50 m × 1 m (50 m²)
  • Search width: 0.5 m on each side of the transect line
  • Survey direction: From end (50 m) to start (0 m) of the transect
  • Timing: Conducted after completing coral recruit quadrats
  • Typical duration: 15-20 minutes per transect

Data Collection Procedure

  1. Deployment coordination:
    • The invertebrate survey utilizes the same 50 m transect line deployed for the LPI survey
    • After completing coral recruit quadrats, the diver transitions to invertebrate surveys
  2. Survey method:
    • Begin at the end of the transect (50 m mark)
    • Search 0.5 m on either side of the transect (1 m total width)
    • Move systematically along the transect toward the start (0 m)
  3. Target taxa:
    • Focus on conspicuous mobile invertebrates (see Target Taxa section)
    • Record species-level identifications when possible
    • For challenging identifications, use higher taxonomic groupings with descriptive modifiers
  4. Data recording:
    • Count individuals by species or lowest identifiable taxonomic level
    • Measure or estimate size for speicfic taxa (length, width, or diameter as appropriate)
    • Note unusual behavior, aggregations, habitat associations, or rare species
  5. Photo documentation:
    • Photograph unusual or unidentified specimens
    • Capture representative individuals of key species

Target Taxa

The invertebrate survey focuses on conspicuous mobile invertebrates that can be reliably observed through visual census. While the specific target taxa may vary by region, the following groups are consistently surveyed:

Echinoderms

Sea Stars (Asteroidea)
Count all individuals, measure diameter
Examples: Acanthaster spp., Linckia spp., Culcita spp.

Sea Urchins (Echinoidea)
Count all individuals, measure test diameter
Examples: Diadema spp., Echinometra spp., Tripneustes spp.

Sea Cucumbers (Holothuroidea)
Count all individuals, measure length
Examples: Holothuria spp., Stichopus spp., Actinopyga spp.

Feather Stars (Crinoidea)
Count all individuals, note color morphs
Examples: Comanthus spp., Oxycomanthus spp.

Mollusks

Giant Clams (Tridacnidae)
Count all individuals, measure shell length
Examples: Tridacna spp., Hippopus spp.

Nudibranchs
Count all individuals, note distinctive species
Examples: Chromodoris spp., Phyllidia spp.

Octopuses and Cuttlefish
Count all individuals, estimate size
Examples: Octopus spp., Sepia spp.

Conch and Large Gastropods
Count all individuals, measure shell length
Examples: Lambis spp., Charonia spp., Cassis spp.

Crustaceans

Lobsters
Count all individuals, measure carapace length
Examples: Panulirus spp., Enoplometopus spp.

Large Crabs
Count all individuals, measure carapace width
Examples: Carpilius spp., Etisus spp.

Special Considerations

Crown-of-Thorns Starfish

In regions where Acanthaster outbreaks are a concern, additional data may be collected, including:

  • Size (diameter) of each individual
  • Feeding activity (active feeding, moving, or stationary)
  • Condition (healthy, injured, or diseased)
  • Aggregation characteristics (isolated or clustered)

Giant Clams

For all Tridacnidae encountered:

  • Record species-level identification
  • Measure maximum shell length (cm)
  • Note substrate type and position (embedded, loose, etc.)
  • Document any signs of harvesting or damage

Commercially Valuable Species

For species targeted by local fisheries:

  • Record precise size measurements
  • Document evidence of harvesting pressure
  • Note habitat associations and aggregation behaviors

Taxonomic Resolution

The invertebrate survey employs the same taxonomic resolution model used across all Pristine Seas benthic methods.

  1. Field Names: Descriptive placeholders used during the survey
  2. Morphotaxa: Consolidation of field names into taxonomic units with distinguishing characteristics
  3. Lowest Defensible Taxonomic Rank: Mapping to formal taxonomy based on available evidence
  4. Functional Group: Classification into ecological functional roles

Data Processing

Data Entry

Invertebrate data are entered into standardized Excel fieldbooks with the following structure:

  • readme sheet: Contains expedition info, data entry instructions, and guidelines
  • stations sheet: Records station information including sampling depths and any deviations from protocol
  • observations sheet: Primary data entry for all invertebrate observations, including counts and measurements

Processing Pipeline

The data processing workflow follows these key steps:

  1. File Consolidation:
    • Merge data across expedition legs and divers
    • Standardize taxonomic names and size measurements
    • Integrate with site metadata from the UVS Sites Fieldbook
  2. Data Validation:
    • Verify completeness of required fields
    • Flag outliers for size measurements
    • Cross-reference taxonomic entries against reference lists
  3. Density Calculation:
    • Calculate abundance per unit area (individuals/m²)
    • Generate summary statistics by taxa, site, and region
    • Create standardized density metrics for comparisons
  4. Database Integration:
    • Format processed data for database ingestion
    • Ensure consistency with other UVS components

Core Output Files

The pipeline generates several essential files:

  • inverts_observations.csv: Cleaned observations with standardized taxonomy
  • inverts_density_by_taxa.csv: Density calculations by taxa and station
  • inverts_station_summary.csv: Aggregate metrics by station
  • inverts_taxa_summary.csv: Occurrence and abundance by taxa across regions

Exploratory Data Analysis

Standard analyses conducted for invertebrate data may include:

  • Community Composition:
    • Taxonomic richness and diversity indices
    • Multivariate analyses (NMDS, cluster analysis)
    • Functional group proportions
  • Statistical Comparisons:
    • Tests for significant differences by region, habitat, and depth
    • Analysis of key indicator species abundance patterns
    • Correlation with benthic cover metrics from LPI
  • Spatial Visualization:
    • Density maps for key taxa
    • Geographic distribution patterns
  • Size Structure:
    • Size frequency distributions for key species
    • Comparative size analysis across protection levels

Limitations

While the belt transect methodology is effective for surveying many mobile invertebrates, several limitations should be considered:

  • Cryptic species: Many invertebrates are cryptic or nocturnal and may be underrepresented in daytime surveys
  • Substrate complexity: Detection probability varies with habitat complexity
  • Diver influence: Some mobile species actively avoid or are attracted to divers
  • Taxonomic challenges: Invertebrate identification often requires specialized expertise
  • Temporal variability: Many invertebrates show significant diurnal or seasonal patterns

These limitations are addressed through:

  • Consistent survey timing and methods
  • Standardized observer training
  • Photographic documentation for verification
  • Conservative taxonomic resolution approach
  • Integration with other survey methods (e.g., eDNA)
